## Development Setup

Dự án này sử dụng [ReactJS](https://react.dev/) / [Next.js](https://nextjs.org/) để tối ưu hóa SEO và trải nghiệm người dùng, chuẩn bị deploy lên [Vercel](https://vercel.com).

*(Note: Install Node.js and run `npm install` and `npm run dev` to start the project once initialized).*
